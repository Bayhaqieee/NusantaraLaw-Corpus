
# Indonesian Constitutional Law (UUD 1945) Corpus

## Overview

This JSON file contains a comprehensive collection of Indonesian Constitutional Law (Undang-Undang Dasar/UUD) articles and provisions from 1945, including the original text and four amendments (1999-2002).

## Contents

The `uud.json` file includes:

- **Preamble (Pembukaan)**: Four foundational paragraphs establishing national values and independence
- **Original Constitution**: 37 articles covering state structure, government powers, and citizen rights
- **First Amendment (1999)**: Changes to presidential powers and legislative procedures
- **Second Amendment (2000)**: Expanded human rights chapter and regional autonomy provisions
- **Third Amendment (2001)**: Presidential election reforms and constitutional court establishment
- **Fourth Amendment (2002)**: Final adjustments to state institutions

## Data Format

Each entry contains:
- `instruction`: Question or topic identifier
- `context`: Section classification (chapter/article heading)
- `response`: Full text of the provision with citations

## Source

**Official Source**: [Indonesian Ministry of Law and Human Rights - JDIH](https://jdih.kemenkum.go.id/dokumen/view?id=1)
